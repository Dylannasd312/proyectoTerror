# juego_terrorUE
repositorio de videojuego de terror desarrollado en unreal engine 5 con bluprints
## Descripción General
MONSTRUO es un videojuego de terror psicológico en primera persona donde el jugador despierta dentro de una casa desconocida y debe encontrar la forma de escapar. A lo largo del juego, el usuario explora el entorno, interactúa con objetos, resuelve mecánicas y descubre una historia oscura mediante notas ocultas.

El juego se centra en la tensión, la exploración y la narrativa ambiental. El jugador no cuenta con armas, por lo que deberá evitar al enemigo y utilizar su entorno estratégicamente para sobrevivir.

## Instrucciones para Ejecutar el Proyecto

1. Clonar el repositorio:
git clone <URL_DEL_REPOSITORIO>

2. Abrir el proyecto con Unreal Engine 5.

3. Ejecutar el proyecto desde el editor:
Presionar el botón Play

4. (Opcional) Ejecutar el build:
Ir a la carpeta Build o Windows  
Ejecutar el archivo .exe
